## 格式
```py
<!doctype html>
  <html lang="en" dir="ltr">
    <head>
    # 网站需要的后台内容
    </head>
    
    <body>
    # 网站内容
    </body>
  </html>
```

## Head  

```py
<head>
  <meta charset="utf-8">
   # <meta> 各种功能
   
  <link href="" rel="stylesheet">
  <link rel="icon/preload" href="">
   # <link>各种功能
   
  <title> browser title </title>
  
</head>

<script src=""> </script> 加载javascript文件，一般在结尾
```


## Body
```py
<main> 只用一次

<header>  标题
<footer>  结尾
<article> 小段文字
<section> 部分
<aside>   旁边的注解
```
