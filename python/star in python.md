# 函数变量中加入星号

## 单星号为传入多个参数转换为元组

```python
def one(a,*b):
    """a是一个普通传入参数，*b是一个非关键字星号参数"""
    print(b)
one(1,2,3,4,5,6)
```

### 输出
```py
(2, 3, 4, 5, 6)
```
## 双星号为转换为字典
```py
def two(a=1,**b):
    """a是一个普通关键字参数，**b是一个关键字双星号参数"""
    print(b)
two(a=1,b=2,c=3,d=4,e=5,f=6)
```
### 输出
```py
{'b': 2, 'c': 3, 'e': 5, 'f': 6, 'd': 4}
```
